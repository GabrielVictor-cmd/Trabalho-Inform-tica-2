Trabalho sobre HTML, CSS e Programação Web

Introdução

A internet faz parte do nosso dia a dia, seja para estudar, trabalhar, fazer compras ou se divertir. Por trás de cada site que acessamos, existe um conjunto de tecnologias que o torna possível. As principais delas são o HTML, o CSS e as linguagens de programação voltadas para a web. Este trabalho tem como objetivo apresentar essas ferramentas de forma simples e clara.

1. O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a base de qualquer site. Ele serve para estruturar o conteúdo da página, como títulos, parágrafos, imagens, links e listas.

2. O que é CSS?

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). Ele é usado para dar estilo às páginas HTML, como cores, fontes, tamanhos e posicionamento dos elementos. O CSS pode ser escrito junto com o HTML ou em um arquivo separado. Ele deixa os sites mais bonitos e agradáveis de usar.
